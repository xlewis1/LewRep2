# AI Use and Scraping Policy 🤖

This policy outlines the acceptable use of Artificial Intelligence (AI) tools regarding code contributions, maintenance, and data scraping for the `lewrep2` repository.

## 1. Code Contributions and Pull Requests

We welcome the use of AI assistants (e.g., GitHub Copilot, ChatGPT, Claude) by human developers to assist in writing code, optimizing performance, or finding bugs, provided the following conditions are met:

* **Human Review Required:** All AI-generated code must be thoroughly reviewed, understood, and tested by the human contributor before submitting a Pull Request. 
* **Licensing Compliance:** Contributors must ensure that any AI-generated code does not violate third-party intellectual property or copy-left licenses.
* **No Blind Pasting:** Code submissions that are clearly unverified, broken, or blindly copied from an AI model will be closed immediately.

## 2. Automated Bots and AI Issues

* Automated AI bots are strictly prohibited from opening issues, formatting pull requests, or commenting on this repository without prior explicit permission from the maintainer (`xlewis1`).

## 3. Data Scraping and Model Training

* **Training Restrictions:** The source code, documentation, and commit history in this repository are protected under the MIT License. While public, automated bulk scraping of this repository specifically for the purpose of training commercial, closed-source machine learning or large language models (LLMs) without attribution is discouraged.
* Proper copyright attribution to `xlewis1` must be preserved if any significant logic or architectural patterns from this repository are ingested into training datasets.
